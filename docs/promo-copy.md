# Excel-Dr 宣传文案

## 一句话介绍

Excel-Dr 是一款专门修复 Excel / WPS 报表卡顿问题的便携工具，能够精准识别并清理隐藏垃圾对象，让那些越用越卡的表格重新变得顺手。

## 短版介绍

很多人以为 Excel 卡，是因为数据太多、电脑太差。

其实很多报表真正的元凶，是 `.xlsx` 里长期积累的隐藏垃圾对象、异常绘图层和损坏关系。它们肉眼看不到，却会让报表打开慢、关闭慢、保存慢、复制后继续卡。

Excel-Dr 做的，就是像医生一样，帮你把这些“看不见的病灶”精准找出来，再像手术刀一样做保守清理。

## 适合发公众号 / 长图说明的版本

### Excel-Dr 是什么

Excel-Dr 是一款面向真实办公场景的 Excel 报表修复工具。

它并不是“表格美化工具”，也不是“万能优化工具”，而是专门针对 Excel / WPS 报表中那些长期积累、肉眼看不到、却严重影响体验的隐藏垃圾数据做精准识别与清理。

### 它解决的是什么问题

很多企业里的财务、运营、订单、仓储、客服、行政岗位，都在长期使用 Excel / WPS 维护业务报表。

这些报表在长期复制、粘贴、继承模板、多人接力编辑后，常常会出现：

- 打开越来越慢
- 保存和关闭越来越卡
- 滚动、筛选、复制时卡顿明显
- 新建的文件也会继承老模板的问题

真正的问题，很多时候并不在可见数据，而在 `.xlsx` 内部的隐藏对象层。

### Excel-Dr 的核心特点

- **精准识别**：重点锁定异常膨胀的隐藏 drawing 对象
- **保守清理**：不粗暴删除正常公式、样式和图片公式
- **上手简单**：普通办公用户也可以直接使用
- **支持批量**：适合部门内部一次性处理一批历史报表
- **便携无安装**：Windows 解压即用，不污染环境

### 它像什么

如果把报表卡顿问题比作“病灶”，那 Excel-Dr 更像一个 Excel 医生。

它不是拿锤子砸，而是像医生手里的手术刀一样，尽量只处理真正异常的部分，保留报表正常的业务结构。

## GitHub 仓库简介文案

Excel-Dr is a lightweight Windows tool that precisely detects and cleans hidden junk objects inside slow Excel and WPS workbooks. It focuses on bloated hidden drawing objects and broken workbook relationships while preserving formulas, styles, and normal cell image content.
