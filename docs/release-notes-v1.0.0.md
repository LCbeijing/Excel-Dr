# Excel-Dr v1.0.0

首个公开版本发布。

## 新增

- Windows 便携版 GUI 工具
- 单文件检测
- 单文件清理并另存为 `_cleaned.xlsx`
- 文件夹批量检测
- 文件夹批量清理
- 清理前确认提示
- 保守的隐藏 `drawing` 对象识别策略
- 损坏 `#REF!` 数据有效性清理

## 特点

- 不覆盖原文件
- 默认保留公式、样式、共享字符串和 `cellImages` 图片公式
- 适合财务、运营、订单、仓储等日常办公报表场景
